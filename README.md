# UNIX V6++ 离散化内存管理

同济大学操作系统课程设计 — 基于 UNIX V6++ 的内存离散化改造。

## 目标

1. **页表体系改造** — 每个进程独立页目录，支持 4M 页表空间
2. **位示图分配** — 以位图方式管理物理页框，替代原连续分配
3. **离散化存储** — 正文/数据/栈段逐页分配，物理页框无需连续
4. **正文段共享** — 相同文件的正文段在进程间共享，只加载一次
5. **写时复制 (COW)** — fork 后父子共享数据/栈页，写入时复制

## 构建

```bash
cd code
make
```

## 项目结构

| 目录 | 说明 |
|------|------|
| `boot/` | 引导扇区 |
| `kernel/` | 内核入口与初始化 |
| `mm/` | 内存管理（页分配器、位图） |
| `proc/` | 进程管理（fork/exec/exit） |
| `fs/` | 文件系统 |
| `dev/` | 设备驱动 |
| `interrupt/` | 中断与异常处理 |
| `pe/` | PE 文件加载器 |
| `tty/` | 终端与键盘 |
| `shell/` | 命令行解释器 |
| `program/` | 用户程序 |
| `test/` | 单元测试 |
